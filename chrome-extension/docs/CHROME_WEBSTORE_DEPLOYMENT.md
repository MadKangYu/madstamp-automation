## 2. 확장 프로그램 패키징

### 2.1 아이콘 준비
`chrome-extension/src/assets/icons/` 디렉토리에 다음 크기의 아이콘 필요:
- `icon16.png` (16x16)
- `icon32.png` (32x32)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

**아이콘 요구사항**:
- PNG 형식
- 투명 배경 권장
- 빨간색 도장 모티브 디자인

### 2.2 스크린샷 준비
웹스토어 등록 시 필요한 스크린샷:
- **최소 1개, 최대 5개**
- **크기**: 1280x800 또는 640x400
- **형식**: PNG 또는 JPEG

권장 스크린샷:
1. 대시보드 화면
2. Gmail 연동 화면
3. Lovart AI 자동화 화면
4. 주문 목록 화면
5. 설정 화면

### 2.3 ZIP 파일 생성
```bash
cd /home/ubuntu/madstamp-automation/chrome-extension
zip -r madstamp-extension-v1.0.0.zip . -x "*.git*" -x "docs/*" -x "*.md"
```
