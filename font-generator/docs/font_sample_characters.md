# 한글 폰트 생성용 샘플 글자 세트

AI 폰트 생성 시 사용하는 샘플 글자 세트입니다. 각 세트는 겹치지 않게 구성되어 있습니다.

---

## 1. 8자 세트 (MX-Font 기본)

한글 자모의 핵심 조합을 포함한 최소 세트입니다.

```
가 나 다 라 마 바 사 아
```

**포함 자모:**
- 초성: ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ
- 중성: ㅏ
- 종성: 없음

---

## 2. 28자 세트 (DM-Font 기본)

한글의 기본 자모 조합을 모두 포함한 세트입니다.

```
갈 같 강 개 걔 거 겨 계
고 과 괴 구 궈 귀 그 긔
기 깨 꺼 꼬 꾸 끼 나 냐
녀 노 뇨 누
```

**포함 자모:**
- 초성: ㄱ, ㄲ, ㄴ (14개 중 일부)
- 중성: ㅏ, ㅐ, ㅑ, ㅒ, ㅓ, ㅕ, ㅖ, ㅗ, ㅘ, ㅙ, ㅚ, ㅛ, ㅜ, ㅝ, ㅞ, ㅟ, ㅠ, ㅡ, ㅢ, ㅣ (21개 중 일부)
- 종성: ㄹ, ㅌ, ㅇ (27개 중 일부)

---

## 3. 43자 세트 (내글네글 기본)

모든 초성과 다양한 중성/종성 조합을 포함합니다.

```
가 각 간 갈 감 갑 강 개
객 거 건 걸 검 겁 게 겨
격 견 결 경 계 고 곡 곤
골 공 과 관 광 괴 구 국
군 굴 궁 권 귀 규 균 그
극 근 글
```

**특징:**
- 모든 기본 초성 포함
- 다양한 받침 조합 포함
- 복모음 일부 포함

---
